# Loan Default Prediction Using Machine Learning

## Overview

This project develops a machine learning pipeline to predict loan default risk using LendingClub loan data. The analysis includes data cleaning, exploratory data analysis, feature engineering, handling class imbalance, model evaluation, and model selection.

## Dataset

* Source: LendingClub
* Records: 1.37 Million+
* Target Variable: Loan Default Status

## Project Workflow

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Class Imbalance Analysis
5. Model Training and Evaluation
6. Model Comparison and Selection

## Models Evaluated

* Logistic Regression
* Random Forest
* LightGBM
* XGBoost

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC

### Best Model

* XGBoost
* AUC: 0.951
* Accuracy: 0.891

## Key Findings

* Compared four machine learning algorithms for loan default prediction.
* XGBoost delivered the strongest predictive performance.
* Reduced prediction errors by approximately 25% compared to baseline approaches.
